## 프로퍼티(property)

### 프로퍼티의 존재 이유

```csharp
class MyClass
{
	int myField;
	public int GetMyField()
    {
		return myField;
    }
	public void SetMyField(int value)
    {
		myField = value; // getter, setter를 따로 구현해야함.
    }
}
```

- 필드에 은닉성을 부여하기 위해 주로 필드를 private 접근자로 제한하고 일반적으로 getter/setter 메소드를 활용함.
